# 阶段1测试指南

## 编译和加载

1. **编译模组**
   ```
   在tModLoader中点击"Workshop" -> "Develop Mods" -> "Build + Reload"
   ```

2. **检查加载**
   - 确认模组在模组列表中显示
   - 确认没有加载错误

## 功能测试

### 1. 启用调试模式

在游戏中打开配置：
- 主菜单 -> Workshop -> Mod Configuration -> Soul Attuned
- 启用 "Enable Debug Mode"
- 重新加载模组

### 2. 测试经验获取

1. 创建新角色和新世界
2. 击杀几个史莱姆
3. 观察聊天框是否显示升级消息
4. 使用命令查看状态：`/soul info`

**预期结果**：
- 击杀怪物后获得经验
- 经验足够时自动升级
- 升级时显示金色消息
- 每级获得1基础点
- 每5级获得1天赋点

### 3. 测试等级上限

1. 使用命令快速升级：`/soul setlevel 10`
2. 尝试继续升级：`/soul addxp 10000`
3. 检查等级是否卡在10级

**预期结果**：
- 未击败任何Boss时，等级上限为10
- 超过上限的经验被丢弃

### 4. 测试谐振阶段

1. 使用作弊物品击败克苏鲁之眼
2. 检查等级上限是否提升到20
3. 继续升级测试

**预期结果**：
- 击败Boss后等级上限提升
- 可以继续升级到新的上限

### 5. 测试基础属性效果

1. 使用命令分配属性：
   ```
   /soul info  # 查看当前状态
   ```
2. 手动修改存档文件或等待UI实现后测试

**注意**：基础点分配需要UI，当前只能通过修改存档测试效果

### 6. 测试数据保存

1. 升级到5级
2. 退出游戏
3. 重新进入游戏
4. 使用 `/soul info` 检查数据

**预期结果**：
- 等级、经验、点数都正确保存
- 重新加载后数据不丢失

## 调试命令

```
/soul info              # 显示当前状态
/soul addxp <数量>      # 添加经验
/soul setlevel <等级>   # 设置等级
/soul addtalent <点数>  # 添加天赋点
/soul addbasic <点数>   # 添加基础点
/soul reset             # 重置所有数据
```

- 天赋解锁需要UI（阶段2）
